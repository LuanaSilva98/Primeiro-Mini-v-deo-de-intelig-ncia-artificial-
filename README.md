# 🎥 Primeiro Mini Vídeo de Inteligência Artificial
Desafio Inteligência artificio Curso Dio 

## 📝 Descrição do Projeto
Este projeto faz parte do desafio do curso da DIO. O objetivo foi utilizar ferramentas de IA Generativa para criar um vídeo curto e demonstrativo.

## 🚀 Tecnologias Utilizadas
* **IA Generativa:** (Meta Ai) para criação do vídeo.
* **GitHub:** Para versionamento e organização do portfólio.

## 📁 Organização das Pastas
* `/Mini video - inteligencia artificial`: Contém o arquivo original da demonstração.

## Processo de Criação 
* Utilizei o site Meta Ai com o seguite Prompt 
"Um mini vídeo de 10 segundos de uma jovem (baseado nas fotos do anexo) estudando banco de dados, sentada em uma mesa com um computador à frente. Ela está digitando códigos SQL no computador, que mostra os códigos na tela. Há vários livros ao redor da mesa. O vídeo deve ser bem realista, preservando as características da jovem."
